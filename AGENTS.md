### Repository overview

This project is called Elden Ring Grace Guesser. Every day users can log in to guess the location on the map of five different graces inside of Elden Ring. They then get a score that they can share with their friends.

## Required Validation After Changes

At the end of every run, run the following commands in order:

1. `pnpm run test`
2. `pnpm run typecheck`

Then summarize changes for the user.

## Preferences

- Do **_NOT_** leave excessive comments when writing code. Only leave comments when
  the code itself does not clearly explain what it does
- Never attempt to start the dev server.
